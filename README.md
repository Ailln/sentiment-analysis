# Deep Learning Initialization Template

🧠 深度学习初始化模板

## 目录简介

```
.
├── README.md          # 简介
├── app                # 应用
│   ├── test.py
│   └── train.py
├── model              # 模型
├── data               # 数据
├── util               # 辅助
│   └── data_util.py
├── save               # 保存
├── config             # 配置
│   └── default.yaml
├── script             # 脚本
│   ├── env.sh
│   ├── test.sh
│   └── train.sh
├── server             # 部署
│   └── app.py
├── test               # 测试
├── tool               # 工具
├── doc                # 文档
├── log                # 日志
├── requirements.txt   # 依赖
└── LICENSE            # 许可
```

## 环境准备

- Ubuntu 18.04 LTS+
- Python 3.6 +
- Anaconda 3
